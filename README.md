# Mobile_Paint

개발 도구 : Android Studio
주 언어 : java

모바일 응용 소프트웨어 기말 과제
 - 각 기능을 가진 버튼과 메뉴 item을 클릭하여 설정을 반영, 그림을 그리는 애플리케이션.
 - 전체 기능 : 직선, 원, 사각형, 곡선, 각지게, 부드럽게, 두께 조절, 색상 선택, 지우기 등
